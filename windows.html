Unique ART Barcode Printer

A Windows Desktop application that interfaces with Zebra ZD420 Barcode Label Printers to print Unique ART Numbers with constraints at facilities and hospitals at pilot sites across the country.

  Download   User Manual   Printer Setup Guide

System Requirements

  • OS: Windows 7 & Above
  • Software and Drivers:
    • Java Runtime Environment 8+
    • Zebra Setup Utilities for Windows
